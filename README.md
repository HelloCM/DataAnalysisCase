### DataAnalysisCase
此项目中主要是收集了互联网上以及各种数据分析书籍中的优秀案例，并结合本人自身的浅薄理解进行复现和注释，环境使用Python3+jupyter-notebook。分享的目的：一方面是方便自己复习查阅，另一方面，也希望能够认识一些志同道合的朋友，一同进步。
